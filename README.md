# 📚 Estrutura de Dados — Exercícios com Arrays em Java

## 📌 Descrição

Este repositório contém a implementação das **Questões 1 a 5 da disciplina de Estrutura de Dados**.  
Os exercícios foram desenvolvidos em **Java** e demonstram operações básicas com arrays, como **inserção, remoção, redimensionamento, eliminação de duplicatas e fusão de vetores ordenados**.

Durante as implementações foram utilizadas técnicas de **deslocamento de elementos (shift left e shift right)** e manipulação de índices para manter a organização dos dados dentro do vetor.

Exemplo de deslocamento utilizado nos algoritmos:

```java
for(int i = size; i > position; i--){
    array[i] = array[i - 1];
}
```

📋 Estrutura do Código

O projeto foi dividido em classes separadas para cada exercício:

Quest01 – Inserção de elemento em posição específica

Quest02 – Redimensionamento dinâmico de array

Quest03 – Remoção de elemento mantendo continuidade

Quest04 – Remoção de elementos duplicados

Quest05 – Fusão (merge) de dois arrays ordenados

Cada classe possui um método main responsável por executar a lógica do algoritmo.
